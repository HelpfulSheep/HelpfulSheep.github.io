Part 01

- download Windows x86-64 executable installer from this page

- install, and make sure to check Add Python 3.5 to PATH

- open up Command Prompt and type

python
print("Hello, world!")
":) " * 100

- to exit the interactive prompt, type Ctrl+Z and press Enter

- create a file called script.py on your desktop

- add some code to the file above i.e.

import random


print("99 little bugs in the code")
print("99 little bugs in the code")
print("Take one down, patch it around")
bug_count = random.randint(100, 200)
message = "{} little bugs in the code".format(bug_count)
print(message)

- in Command Prompt, type

cd Desktop
python script.py

- congrats, you just ran your first Python script! :)
