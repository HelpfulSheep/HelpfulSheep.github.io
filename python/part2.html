Part 02

- open up Command Prompt and type

pip install requests
pip install beautifulsoup4
pip install html5lib==1.0b8

- create a Python script file with the following code, and run it

import bs4
import requests


response = requests.get("http://helpfulsheep.com/python/part2.html")
soup = bs4.BeautifulSoup(response.text, "html5lib")
title = soup.find("h1")
print(title.text)

- congrats, you just ran your first Python scraper! :)
